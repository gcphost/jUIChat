jQuery UI Live Chat (juichat)

  • Using jQuery and jQuery UI for themeing
  • Written in PHP using MySQL
  • Designed to be like: Gmail/Facebook Style jQuery Chat (but 100% free!)
  • jQuery CSS3 Emoticons
  • Licensed under LGPL (thinking MIT)

Example Usage:

<script type="text/javascript"> 
	$().livechat({'getViewers': true}); 
</script>

Options:

  • recheckdelay: delay in time to check for new chats/viewers
  • idletime: delay in time before a user is idle (yellow)
  • getViewers: show the viewers
  • viewerwrapper: the class name or div for viewer list
  • emoticons: enable emoticons
  • page: the php page
